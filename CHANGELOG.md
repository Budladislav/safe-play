# История изменений Safe Play

Все заметные изменения приложения фиксируются здесь. Версии следуют схеме `MAJOR.MINOR.PATCH`.

## Unreleased

- Зафиксирован постоянный локальный workflow разработки и релизов.

## [2.1.0] — 2026-07-21

### Добавлено

- Редактирование и удаление завершённых сессий с автоматическим пересчётом статистики.
- Статистика игрового времени по неделям за последние 8 недель.
- Быстрый PNG-отчёт за текущую неделю: скачивание и системное меню «Поделиться» для отправки в мессенджер.
- Возможность выбрать несколько причин желания играть.
- Смысловые подписи всех пяти значений под каждым ползунком оценки.
- Видимый номер версии и ссылка на этот ченджлог в настройках.

### Изменено

- Формат локальных данных обновлён до схемы 3. Старые сессии с одной причиной автоматически совместимы с новым множественным выбором.
- Офлайн-кэш обновлён для нового релиза.

## [2.0.0] — 2026-07-20

Первый опубликованный MVP Safe Play:

- офлайн-PWA без аккаунта и бэкенда;
- библиотека игр;
- настраиваемый чек-лист входа и осознанный override;
- планирование сессии, действие после игры и таймер на временных метках;
- продления с причиной и лимитом;
- короткая процедура выхода;
- история, тепловая карта, план/факт и базовая статистика контроля;
- мягкий cooldown;
- полный экспорт и импорт JSON;
- публикация через GitHub Pages.
