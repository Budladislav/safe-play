# Safe Play

Одностраничное офлайн-PWA для контролируемого гейминга: короткая пауза до старта, заранее выбранные границы и факты после сессии.

## Запуск

```bash
npm run dev
```

Затем откройте `http://127.0.0.1:4173`. Для установки на Android приложение нужно отдать по HTTPS (локальный `localhost` считается безопасным контекстом только на том же устройстве).

## Проверка

```bash
npm test
npm run check
```

Данные хранятся в `localStorage` под ключом `safe-play:v2` (схема данных 3). Экспорт и импорт полной JSON-копии находятся в настройках.
